# 🛡️ Sistema Web para la Gestión y Consulta de Información del Personal

Sistema web desarrollado para la **1ra Brigada de Servicios del Ejército del Perú**, orientado a la gestión y consulta de información del personal militar y civil mediante una plataforma centralizada con asistente virtual basado en Inteligencia Artificial.

## ✨ Funcionalidades

- Inicio de sesión con verificación mediante correo electrónico.
- Gestión de usuarios.
- Registro de personal militar.
- Registro de personal civil.
- Consulta de información del personal.
- Actualización y eliminación de registros.
- Generación de reportes en PDF y Excel.
- Visualización de gráficos estadísticos.
- Asistente virtual con Inteligencia Artificial (Groq).
- Gestión centralizada de la información institucional.

## 📌 Observaciones

El sistema se encuentra desplegado en un servidor web mediante **Render**, por lo que **no requiere instalación local**. Para utilizarlo únicamente es necesario acceder mediante el enlace proporcionado e ingresar las credenciales autorizadas.
